Projet - Kasa

Dans ce projet, j’implémente le front-end d’une application en utilisant React et React Router pour créer une expérience utilisateur moderne et réactive. Je travaille sur la logique de présentation des données et les composants React, tout en configurant la navigation entre les pages avec React Router. Le projet se base sur des maquettes fournies et utilise des données simulées à partir d’un fichier JSON. J’ai commencé avec Create React App pour simplifier la configuration initiale et j’utilise Node.js pour exécuter du code en dehors du navigateur. Ce projet me permet de renforcer mes compétences en React, React Router, et d’implémenter des animations CSS pour améliorer l’interface utilisateur.

Technologies utilisées :

React - Sass

Problématiques rencontrées

L'une des principales problématiques rencontrées a été de gérer la navigation entre les différentes pages de l'application avec React Router tout en maintenant une expérience fluide et réactive. Il a fallu également optimiser le rendu des composants React pour éviter les rechargements inutiles et améliorer la performance générale de l'application.

Compétences développées

Ce projet m'a permis de renforcer mes compétences en React, notamment la gestion de la navigation avec React Router. J'ai également développé des compétences en optimisation des composants React pour améliorer les performances et en implémentation d'animations CSS pour une meilleure expérience utilisateur.

Chaque site est conçu pour raconter une histoire visuelle unique et captivante.
