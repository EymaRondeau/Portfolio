Projet - Sophie Bluel

Dans ce projet, j’ai créé une page web dynamique pour une architecte d’intérieur en utilisant JavaScript et une API. J’ai développé la page de présentation des travaux à partir d’un HTML fourni, géré les événements utilisateurs et manipulé le DOM. J’ai aussi créé une page de connexion pour l’administrateur et une modale pour l’upload de nouveaux médias. Cela m’a permis d’améliorer mes compétences en JavaScript et d’apprendre à communiquer avec une API, des éléments clés pour le développement web moderne. J’ai utilisé Figma pour les maquettes, Visual Studio Code pour le développement et GitHub pour la version.

Problématiques rencontrées

L'un des défis majeurs dans ce projet était de gérer la communication entre le front-end et l'API de manière efficace et sécurisée. La manipulation du DOM et la gestion des événements utilisateurs ont également nécessité une réflexion approfondie pour assurer une expérience fluide et intuitive pour les utilisateurs.

Compétences développées

Ce projet m’a permis de renforcer mes compétences en JavaScript, notamment dans la gestion des événements et la manipulation du DOM. De plus, j’ai appris à travailler avec des API et à gérer les interactions front-end/back-end, ainsi qu’à améliorer mes compétences en développement web dynamique.

Chaque site est conçu pour raconter une histoire visuelle unique et captivante.
